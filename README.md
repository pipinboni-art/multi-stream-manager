
# Multi Stream Manager (YouTube Multi Live)

Features:
- Start many live streams at once
- Each stream has its own input (URL or File)
- Each stream has its own YouTube RTMP output
- Stop individual stream
- View list of active streams

## Deploy on Railway (FREE)
1. Go to https://railway.app
2. Create a new project → Deploy from ZIP
3. Upload this ZIP
4. Railway builds automatically
5. Open your Public URL → app ready

## API
POST /start
POST /stop
GET /list
